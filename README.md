# Florune 🌸🎧

**Florune** is a lightweight and experimental music player built for **Navidrome** servers.
The project focuses on providing a simple, modern and personal listening experience.

⚠️ **Project Status**

Florune is currently in a **very early Alpha stage**.
This means the project is still at the beginning of development and many features are incomplete or may change.

Expect:

* Bugs
* Missing features
* Frequent updates
* Breaking changes

Florune is being developed as an experimental client for **Navidrome**, aiming to explore new ideas for a clean and enjoyable music player.

## Features (Early)

* Connect to a Navidrome server
* Stream music from your personal library
* Simple and minimal interface
* Experimental features under development

## Warning

This project is **not ready for production use**.
Use it only for testing and development purposes.
